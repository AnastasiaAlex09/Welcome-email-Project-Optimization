# 🚀 Проект оптимизации Welcome-рассылки

## О проекте
Проект направлен на повышение Open Rate и Activation Rate за счет персонализации тем писем и смены отправителя.

## Статус
✅ В разработке

## Цели проекта
- Повысить Open Rate с 18% до 25%
- Увеличить CTR с 2.5% до 4.0%
- Повысить Activation Rate с 12% до 18%

## Структура репозитория

## 📋 Описание основных папок

| Папка | Назначение |
|-------|------------|
| **`.github/`** | Автоматизация через GitHub Actions |
| **`docs/`** | Вся проектная документация |
| **`content/`** | Тексты писем, темы, прехедеры |
| **`design/`** | Макеты, изображения, гайдлайны |
| **`templates/`** | Исходники (MJML) и скомпилированный HTML |
| **`analytics/`** | Данные A/B тестов, метрики, отчеты |
| **`scripts/`** | Утилиты для автоматизации |
| **`tests/`** | Тестирование совместимости и качества |
