# 🚀 Проект оптимизации Welcome-рассылки

## О проекте
Проект направлен на повышение Open Rate и Activation Rate за счет персонализации тем писем и смены отправителя.

## Статус
✅ В разработке

## Цели проекта
- Повысить Open Rate с 18% до 25%
- Увеличить CTR с 2.5% до 4.0%
- Повысить Activation Rate с 12% до 18%

# 🎨 Style Guide для Welcome-писем

## Цветовая палитра

### Основные цвета
- **Primary**: `#4F46E5` (Индиго) - кнопки, важные элементы
- **Secondary**: `#10B981` (Изумрудный) - успешные действия
- **Accent**: `#F59E0B` (Янтарный) - выделения, бейджи

### Нейтральные цвета
- **Text Primary**: `#111827` - основной текст
- **Text Secondary**: `#6B7280` - вторичный текст
- **Background**: `#F9FAFB` - фон письма
- **Border**: `#E5E7EB` - границы, разделители

## Типографика

### Шрифты
- **Основной**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Helvetica, Arial
- **Fallback**: sans-serif

### Размеры
- **Заголовок H1**: 24px, bold
- **Заголовок H2**: 20px, semi-bold
- **Основной текст**: 16px, regular
- **Вторичный текст**: 14px, regular
- **Мелкий текст**: 12px, regular
